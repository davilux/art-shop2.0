## Art Shop

# Technologies:

- React
- Redux
- Sequelize
- Express
- Webpack
- Babel

When cloning to a new device, you will need to:

- `npm i`
- `touch .env`
- add values for SECRET_KEY and REFRESH_SECRET_KEY to .env
- `createdb art-shop`
- `npm run seed`
- `npm start`
- open localhost:3000 in browser to view the site
